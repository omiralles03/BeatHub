# BeatHub

BeatHub is a Node.js application built using Express, Passport (with OAuth2), and TypeScript.

## Prerequisites

- **Node.js** (v14.x or later recommended)
- **npm** (v6.x or later)

## Installation

1. **Clone the Repository**

    ```bash
    git clone https://github.com/omiralles03/BeatHub
    cd BeatHub
    ```

2. **Install Dependencies**

    Production Dependencies:
       - dotenv
       - express
       - express-session
       - passport
       - passport-oauth2

    Development Dependencies:
       - @types/express, @types/express-session, @types/passport, @types/passport-oauth2
       - nodemon
       - ts-node
       - typescript

3. **Running the Application in Dev Mode**

    ```bash
    npm run dev
    ```
